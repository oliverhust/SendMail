# SendMail Ver 2.0.0

Windows和Mac平台的群发邮件工具。支持多账号轮流发送（降低被拒的可能）。支持丰富的邮件编辑功能、进度恢复等

邮件列表：一个Excel表格文件，里面可以有多个工作簿（Sheet），待发送的邮箱统一填在某一列（比如C列）
